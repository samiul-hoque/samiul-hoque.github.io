## Assignment

Develop a plan for dissemination of your final project; investigate intellectual property.

## Context

Self-taught since 2013. The plan: publish PixelFace on Instructables as a contest entry, and maintain the design files on GitLab.

## License chosen — CC BY-SA 4.0

Creative Commons Attribution-ShareAlike 4.0 International.

Allows: modification and commercial use.

Requires: attribution to the original author, and any derivatives must use the same license terms.

This felt right for an open-hardware project: permissive enough to encourage reuse, but the share-alike clause ensures improvements stay in the commons.

## License landscape overview

| License | Key characteristic |
|---|---|
| Patent | Limited-period exclusivity; requires public disclosure |
| MIT | Permissive, minimal restrictions, highly compatible |
| GPL (copyleft) | Freedom to use/study/modify/share; derivatives must use GPL |
| LGPL | Like GPL but allows proprietary linking; suited to libraries |
| BSD | Minimal restrictions on use and redistribution |
| Apache | Requires copyright notice and disclaimer preservation |
| CC BY-SA | Attribution + share-alike; the one chosen for PixelFace |

## Dissemination plan

- Instructables — detailed build guide, contest submission
- GitLab repository — design files, firmware, schematics
- License: CC-BY-SA 4.0 on all materials
