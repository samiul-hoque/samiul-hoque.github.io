## Assignment

Model a possible final project using raster, vector, 2D, and 3D tools; explore at least two new tools.

## 2D raster

**MS Paint** — quick pixel face concept sketch. Useful for rapid ideation.

**GIMP** — free and cross-platform. Used the BIMP plugin for batch image processing. More capable than MS Paint for anything beyond rough sketches.

**Adobe Photoshop** — pen tool, selection, fill. Used for cleaning up the PixelFace concept renders.

## 2D vector

**Adobe Illustrator** — created a laser-cutting design for an early PixelFace mock-up. Exported as DXF (AutoCAD 2004 format) for physical cardboard assembly.

## 3D — Fusion 360

Designed the motor mount for the PixelFace spinning arm:

1. Rough sketch with concentric circles
2. Set parameters: wall thickness, base diameter, motor diameter
3. Extrude
4. Extrude from offset — for the motor mount cavity
5. Export for MeshMixer

Also explored three useful modelling operations post-review (March 26):
- **Draft** — angled surfaces for mold release
- **Fillet** — curved corners for aesthetics and strength
- **Press/Pull** — face deformation for quick shape iteration

## 3D — MeshMixer

Applied the "Make Pattern" tool with a hex grid to the Fusion 360 output. Produces generative-design-like structures with good material efficiency. Fast to iterate on organic patterns.

## Learning outcomes

GIMP proficiency, Fusion 360 basics, MeshMixer. Future: Rhino, Grasshopper.
